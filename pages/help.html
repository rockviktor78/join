What is Join?

Join is a kanban-based project management tool developed by a group of motivated students during their web development bootcamp at the Developer Akademie.

Kanban is a method that focuses on visualizing work and tracking progress in a clear and structured way. Join applies these principles to provide an intuitive interface that helps users plan tasks, monitor progress, and stay organized throughout a project.

Please note that Join is an educational project. It was created for learning purposes and is not intended for professional or large-scale business use. While we aim to provide a smooth user experience, we cannot guarantee permanent availability, reliability, or complete accuracy.

How to use Join

Getting started with Join is simple. The following steps will guide you through the basic features.

The Board Overview

After logging in, you will see a default kanban board that represents your project. The board consists of four columns:

  • To Do
  • In Progress
  • Await Feedback
  • Done

Each column represents a different stage of your tasks.

Managing Contacts

Join allows you to collaborate with others by adding contacts. To create a new contact, go to the Contacts section, click on New contact, and enter the required information. Once added, contacts can be assigned to tasks and take part in the project workflow.

Creating Tasks

Tasks in Join are displayed as cards on the board. You can add a new task by clicking the "+" button in the appropriate column. Each card can include details such as the task title, description, due date, and assigned contacts.

Updating Task Status

As work progresses, tasks can be moved between columns using drag and drop. This makes it easy to track progress and keep the board up to date.

Completing or Deleting Tasks

When a task is finished, move it to the Done column. If a task is no longer needed, it can be deleted. Please note that deleted tasks are removed permanently and cannot be restored.

Final Notes

Join works best when tasks are updated regularly. Keeping the board current ensures that everyone involved has a clear understanding of the project's status.
