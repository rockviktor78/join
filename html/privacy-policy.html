When using our application, personal data is processed to provide the functions of the platform.

Categories of Data Processed

  • Contact data
  • Content data (e.g., tasks, descriptions)
  • Usage and log data
  • Meta and communication data
  • Uploaded files (especially images)

Data Subjects

  • Users of the application
  • Communication partners

Purposes of Processing

  • Provision and operation of the online service
  • Management of tasks and file attachments
  • User authentication
  • Ensuring security and stability
  • Communication with users

4. Security Measures

To protect data, appropriate technical and organizational measures are employed, in particular:

  • Encrypted data transmission (TLS/SSL)
  • Access protection through authentication
  • Role-based permissions
  • Use of a secure cloud infrastructure

5. Processing of Uploads and Attachments

Local Processing

Uploaded files are initially processed in the browser:

  • Automatic scaling of images to a maximum of 800 Ã— 800 pixels
  • Compression to approximately 70% quality
  • Conversion to a suitable transfer format

Cloud Storage

The data is transmitted encrypted to Firebase Storage (Google Cloud Platform, location Europe) and stored there. In this process:

  • Transfers are secured via HTTPS
  • Data is encrypted at rest
  • Files are uniquely assigned to a task
  • Access is enabled only for authenticated users

Access Control

  • Authentication via Firebase Authentication
  • Blocking of unauthorized access
  • Protected download URLs

6. Deletion and Retention Period

  • When a task is deleted, all associated attachments are immediately and permanently removed.
  • Users can delete individual files themselves at any time.
  • No additional archiving of deleted data is performed.

7. Technical Framework

Supported file formats include in particular:

  • JPEG, PNG, GIF, WebP

Firebase or Google Cloud Platform is used as a data processor in accordance with Art. 28 GDPR.

  • Right to lodge a complaint with a supervisory authority
